# Scripture Versions
### Teaching with Clarity in Scripture

---

# What Changed?

From Church Handbook 38.8.40.1:

- Use a **preferred Church Bible** in classes
- Other translations **may also be used**
- Especially when they are:
  - Easier to understand
  - Doctrinally clear

---

# What are my options?

By reading level, these translations are:
  
  3rd grade:
  - New International Reader's Version (NIrV)

  6th through 8th grades:
  - New International Version (NIV)
  - New Living Translation (NLT)
  - New King James Version (NKJV)

  9th through 11th grades:
  - English Standard Version (ESV)
  - New Revised Standard Version (NRSV)

link: [Bible App](https://www.youversion.com/bible-app)

---

# What This *Is* & *Is Not*

This **is**:

- Permission to increase understanding
- An invitation to teach more effectively
- A way to help youth engage with scripture

<br>

This is **not**:

- Replacing the King James Version
- Teaching from random translations
- Changing doctrine

The KJV remains our **foundation** in English-speaking settings.

---

# Why This Matters

Some students struggle with:

- Archaic language
- Misunderstood words
- Losing the message in the wording

If they do not understand it, they cannot apply it.

---

# The Teacher's Role

As teachers, we now can:

- Use another translation to clarify meaning
- Compare wording to deepen understanding
- Teach students how to study, not just read

---

# Example 1 - Luke 14:26 (Discipleship)

```text
+----------------------------------+    +----------------------------------+
|               KJV                |    |               NLT                |
+----------------------------------+    +----------------------------------+
| "If any man come to me, and      |    | "If you want to be my disciple,  |
| hate not his father, and mother, |    | you must, by comparison, hate    |
| and wife, and children, and      |    | everyone else--your father and   |
| brethren, and sisters, yea, and  |    | mother, wife and children,       |
| his own life also, he cannot be  |    | brothers and sisters--yes, even  |
| my disciple."                    |    | your own life. Otherwise, you    |
|                                  |    | cannot be my disciple."          |
+----------------------------------+    +----------------------------------+
```

Potential Misunderstanding:
Jesus commands literal hatred

Clarified Meaning:
Christ must come first in priority
